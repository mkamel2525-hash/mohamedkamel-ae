# Mohamed Kamel — Dubai Luxury Property Advisor

An award-winning luxury personal brand website positioning **Mohamed Kamel** as a
**Private Property Advisor & Portfolio Management Specialist** for Dubai & Abu Dhabi —
not as a traditional broker.

> Off-Plan • Luxury • Investment • Portfolio Management — AED 100M+ Career Sales

## Design

- **Aesthetic:** Luxury editorial (≈80% light luxury / 20% dark luxury contrast)
- **Style system:** Exaggerated Minimalism + Editorial Grid (via the `ui-ux-pro-max` skill — see `design-system/mohamed-kamel/MASTER.md`)
- **Typography:** Cormorant Garamond (headings) · Montserrat (body)
- **Palette:** Warm White `#FFFFFF` · Soft Beige `#FAF7F3` · Warm Cream `#F4EFE8` · Luxury Gold `#C9A96E` · Deep Contrast `#1D1A17` · Text `#2C2420` · Soft Text `#7A6E68`
- **Motion:** Smooth scroll reveal, parallax, floating glass cards, magnetic CTAs, animated counters, interactive process timeline — all gated behind `prefers-reduced-motion`.

## Structure

`index.html` is a single-page experience with 11 sections:
Hero · Trust Bar · About · Services · Market Intelligence · Portfolio Management (dark) ·
Investment Process · Developer Expertise · Trusted by Investors (dark) · Consultation · Footer.

```
index.html          Markup + SEO meta + JSON-LD (Person schema)
css/styles.css      Full luxury design system
js/main.js          Reveal, parallax, counters, magnetic buttons, dev cards, timeline, form
assets/             Portrait images
robots.txt          Crawler directives
sitemap.xml         Single-page sitemap
CNAME               GitHub Pages custom domain (mohamedkamel.ae)
design-system/      Design system generated by the ui-ux-pro-max skill
```

## The portrait

The supplied studio portrait had its grey background removed programmatically
(`assets/mohamed-kamel.png` is a transparent cutout) so it layers cleanly over the
Dubai-skyline hero. The original is kept at `assets/mohamed-kamel-original.png`.

**To replace the photo:** drop a new transparent PNG at `assets/mohamed-kamel.png`
(and the `-about` / `-consult` copies). For best results use a professionally retouched
cutout, warm editorial tones.

The hero background uses a Dubai skyline image. To self-host it instead of the CDN URL,
save your image to `assets/dubai-skyline.jpg` and update `.hero__bg` in `css/styles.css`.

## Consultation form

The form is backend-free: on submit it composes a pre-filled WhatsApp message to
`+971 58 880 1766`, so enquiries arrive instantly with no server required. To route to a
backend or email service instead, edit the submit handler in `js/main.js`.

## Deploy

Pure static site — host anywhere.

**GitHub Pages:** enable Pages on this branch; the `CNAME` file points it at `mohamedkamel.ae`.
**Netlify / Vercel / Cloudflare Pages:** deploy the repo root, no build step.
